# 实现思路
- 编码表 依据特定文件生成编码表,设置Huffman编码并保存到文件
  - 读入文件,统计字符概率
  - 设置Huffman编码表
  - 保存编码表,便于重复利用
- 文本编码 读入待编码文件.进行huffman编码.写入文件 
  - 逐一字符读入
  - 依据Huffman树获得编码,
  - 保存编码,再次读入下一个字符,直到文件结束
- 编码文件译码.读入编码文件,译码,写入文件.显示译码结果或内容.
  - 逐一字符读入
  - 索引表,直到获得一个字符
  - 读到文本结束
- 编码压缩
- 解压缩
